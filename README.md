MasterMind
==========

A classic Master Mind game written in Java
<br/><br/>
Special features:
<ul>
  <li>Intuitive GUI</li>
  <li>Supports 2 players</li>
  <li>Supports playing with Computer</li>
  <li>Multiple rounds</li>
  <li>4 difficulties</li>
  <li>Supports Java applet to be run on web browser</li>
</ul>
